## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- pnpm
- Deployed on Vercel

## Getting Started

Clone the repository:

```bash
git clone https://github.com/raytorresch/raytorres-portfolio.git
cd raytorres-portfolio
````

Install dependencies:
```bash
pnpm install
````

Run the development server:
```bash
pnpm dev
````

Open http://localhost:3000 in your browser.

## Production Build
```bash
pnpm build
pnpm start
````
